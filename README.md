# douban-workflow

项目 fork 自 https://github.com/h3l/douban-workflow, 感谢 @h3l

做了一点小改动
1. 用豆瓣网页版搜索（主要是想要一些演员信息和导演信息等）
2. 添加了音乐搜索

支持搜索书籍以及电影, 音乐。

* `dbb` (douban book 简称)
* `dbm`(douban movie 简称)
* `dbu` (douban music)
